# 🚀 n8n Lead Automation Pipeline

Automação completa de captura e gestão de leads usando n8n.

## 🔥 O que esse workflow faz

Este fluxo automatiza todo o processo de entrada de leads:

✅ Recebe dados via Webhook  
✅ Organiza os dados automaticamente  
✅ Salva em Google Sheets  
✅ Envia e-mail automático ao lead  
✅ Notifica em tempo real via Discord  

---

## 🧠 Tecnologias usadas

- n8n
- Google Sheets
- Gmail API
- Discord Webhooks

---

## ⚙️ Como usar

1. Importe o arquivo `workflow.json` no n8n  
2. Configure suas credenciais:
   - Google Sheets
   - Gmail
3. Adicione seu webhook do Discord  
4. Execute o workflow
